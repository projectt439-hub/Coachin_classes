# Tech Coaching Platform

A modern, Allen-inspired online coaching website for technology courses built using HTML, CSS, and JavaScript.

## Features

- **Professional Design**: Inspired by Allen coaching institute
- **Hero Section**: Engaging landing with key features
- **Technology Courses**: SQL, PostgreSQL, AI, ML, Java, Python
- **Success Stories**: Student achievements showcase
- **User Authentication**: Login/Register system
- **Interactive Learning**: Dynamic course content
- **Responsive Design**: Mobile-friendly interface

## Courses Offered

1. **SQL Database** - Database fundamentals and queries
2. **PostgreSQL** - Advanced database management
3. **Artificial Intelligence** - AI concepts and applications
4. **Machine Learning** - Hands-on ML with Python
5. **Java Programming** - Complete Java development
6. **Python Programming** - Python for all skill levels

## Key Sections

- **Hero Section**: Course highlights and call-to-action
- **Course Grid**: Modern card-based course display
- **Success Stories**: Student achievements and testimonials
- **User Dashboard**: Personalized experience after login

## Technical Features

- **Modal Authentication**: Clean login/register dialogs
- **Local Storage**: Client-side user management
- **Dynamic Content**: Interactive course exploration
- **Modern CSS**: Gradients, animations, grid layouts

## Files

- `index.html`: Main webpage with Allen-inspired layout
- `style.css`: Modern CSS with professional styling
- `script.js`: Interactive features and authentication
- `1000132112.jpg`: Logo with "Tech Coaching" text overlay
- `README.md`: Documentation

## How to Run

```bash
python3 -m http.server 8000
```

Visit http://localhost:8000
